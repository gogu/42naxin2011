行政助理

部门介绍

行政是一个从一开始就伴随小组成长的部门，她可以将小组这个大家庭中杂乱无章的事物管理得井然有序，她协助各部门的工作，管理小组内部的日常事务，协调各部门之间的关系，她是小组不可缺少的部门。

纳新要求

工作职责：

建立、维护、优化42Team工作流程，负责小组日常工作安排、文档记录、财务管理、人力资源分配以及小组任务的通知工作

基本技能和素质要求：

  1. 1．文笔好,性格开朗, 人际沟通能力强,吃苦耐劳
  2. 2．熟练使用OFFICE办公软件
  3. 3．工作效率高，条理性强，有团队合作精神
  4. 4．做事稳重，责任心强，能积极主动完成工作

如果你满足了上述要求就说明你已经很优秀了，但是如果还具有下面的任何一项技能会更好：

  1. 1．熟悉foxmail或其他邮件工具的使用
  2. 2．在学校其它组织有过相关工作经验

前台设计

部门介绍

前台是小组里最具视觉感的部门，网页的制作、配色及框架的设计以及宣传海报的喷绘制作都是他们的拿手好戏。独特的创意和灵活的艺术表现力，在依托丰富的编码经验，让他们充分展示着自己的才华。

纳新要求

工作职责：

让42Team出品的东西（例如：网页，招贴以及周边产品等）漂漂亮亮的，有特色

【—平面设计】基本技能和素质要求：

  1. 1．准确把握项目需求，有独特的创新意识和灵活的艺术表现力
  2. 2．熟悉Photoshop, Illustrator, FIREWORKS, FLASH一种或以上软件

如果你满足了上述要求就说明你已经很优秀了，但是如果还具有下面的任何一项技能会更好：

  1. 1．精通Photoshop, Illustrator, FIREWORKS, FLASH其中的一种软件
  2. 2．有好的手绘/摄影功底
  3. 3．具有网页美术设计或海报喷绘制作经验

【—前端代码】基本技能和素质要求：

  1. 1．熟悉TABLE或(X)HTML+CSS布局至少一种
  2. 2．熟悉DREAMWEAVER或同类编辑器
  3. 3．熟悉HTML语言，有静态网页设计制作经验

如果你满足了上述要求就说明你已经很优秀了，但是如果还具有下面的任何一项技能会更好：

  1. 1．熟悉关于Web标准、易用性、浏览器端原理以及网站性能等方面的知识
  2. 2．熟悉JavaScript脚本语言，了解AJAX应用
  3. 3．关注前端技术新趋势，善于学习，乐于探索，不墨守陈规

后台编程开发部

部门介绍

后台部门是小组里逻辑思维能力最强的部门，关乎着网站功能的强大与否，通过代码连接着创意与实现，将抽象转换成具象。是网站开发的灵魂。

纳新要求

工作职责：

维护现有的J2EE及PHP产品并根据需求完成二次开发，同时开发新应用，新产品

基本技能和素质要求：

  1. 1．你不必能言善辩 ，但是你的品行必须端正、道德必须高尚 、乐于助人
  2. 2．你不必每天都在办公室,但是能够在有需要的时候迅速聚集到一起
  3. 3．你不必要一鸣惊人，但是你必须踏实认真、刻苦钻研 、持之以恒
  4. 4．你不必技术高超（42会把你变成那样），但是你必须有超强的自学能力和一颗追求技术的心
  5. 5．不要好高骛远,对所学的专业知识能够做到向上延伸、向下深入、横向扩展

如果你满足了上述要求就说明你已经很优秀了，但是如果还具有下面的任何一项技能会更好：

  1. 1．熟悉JAVA，C/C++,C#,VB等一种或一种以上开发语言
  2. 2．熟悉ASP/ASP.NET, PHP, JSP, JS等一种或一种以上网站开发语言
  3. 3．熟悉MySQL，SQL Server等一种或一种以上数据库
  4. 4．拥有具体实际项目开发经验

系统维护

部门介绍

系统是小组的核心，是42网的支柱，承担着最大的责任。她搭建服务器并维持服务器的正常运行，为网站的后台运转提供平台，备份网站数据，对服务器的各种服务做优化，偶尔还要跟黑客打交道，时刻注意系统的状况，阻止黑客的入侵。装系统什么的只是最最基本的技能！

纳新要求

工作职责：

通过对操作系统及服务的调配，保证Window及Linux平台服务器的稳定运行，处理现有服务发生的故障，为新服务搭建平台；封堵服务器漏洞；协助数字化教学支持中心维护校园网络

基本技能和素质要求：

  1. 1. 虚心好学，勤奋，能吃苦耐劳，有较强的学习能力（重要）
  2. 2. 熟练操作windows，能够优化系统，了解系统运行机制
  3. 3. 有一定的硬件、网络和安全知识

如果你满足了上述要求就说明你已经很优秀了，但是如果还具有下面的任何一项技能会更好：

  1. 1．熟练配置IIS,APACHE,TOMACAT一种或以上WEB服务器
  2. 2．熟练配置FTP服务器
  3. 3．熟悉LINUX的高级应用
  4. 4．熟悉数据库管理
  5. 5．熟悉操作系统原理
  6. 6．熟悉网络知识
